# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository Purpose

A shared BibTeX bibliography (`master.bib`) for repression studies, broadly defined. Managed with BibDesk.

## Citation Key Convention

Format: `lastname1stauthor:yearFirstWordOnTitle` (all lowercase). First word excludes articles (the, a, an).

- 1 author: `liu:2022dissent`
- 2 authors: `liu:sullivan:2020and`
- 3+ authors: `kim:etal:2023spatial`

## BibTeX Entry Rules

**Omit**: `doi={}`, `url={}` (except working papers), `abstract={}`, `note={}`, PDF links

**Author names**: Full names required with periods on initials — `Michael D. Ward`, not `M.D. Ward`

**Journal names**: Full name, all words capitalized — `Journal of Peace Research`, not `J. of Peace Res.`

**Titles**: Wrap all fields in `{}`; journal article titles in CamelCase; book titles in Title Case

**date-added / date-modified**: BibDesk auto-manages — do not edit manually
